url：/scenewordview/api/edit_node 
请求方式：post 
数据格式：json 
备注: 测试编辑节点， 编辑子节点, node_id 不存在则新增
        请求参数：
        {
          "parent_id": int 必须 // 父亲节点id
          "node_id": int 必须 // 节点id
          "name": str 必须 // 节点名称
        }
        返回参数：
            {
          "status": 1,
          "msg": "success",
          "data": {
            "Name": ,
            "level": 2,
            "ProjectId": 1,
            "RelateEffectIdList": // 被关联节点id列表
            "PicUrl": // 图片地址
            "Path": "/1/2",
            "Id": 2
          },
          "result": 1
        }

{
