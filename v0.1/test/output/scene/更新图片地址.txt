url：/scenewordview/api/update_path 
请求方式：post 
数据格式：json 
备注: 更新图片地址
        请求参数：
        {
          "path": str 必须 // 图片地址
          "node_id": int 必须 // 节点Id
        }
        返回参数：
        {
          "status": 1,
          "msg": "success",
          "data": {
            "Name": ,
            "level": 2,
            "ProjectId": 1,
            "RelateEffectIdList": // 被关联节点id列表
            "PicUrl": // 图片地址
            "Path": "/1/2",
            "Id": 2
          },
          "result": 1
        }
 
例子: 
请求格式: 
{
  "path": "media/word_view/struct/20171106/7b17f803-1b4c-4868-9a9a-1473c4c820de.png", 
  "node_id": 2
} 
返回格式：
{
  "status": 1, 
  "msg": "success", 
  "data": {
    "Name": "深圳的", 
    "level": 2, 
    "ProjectId": 1, 
    "RelateEffectIdList": "12,12,685,684,681,680,683,682", 
    "PicUrl": "media/word_view/struct/20171106/7b17f803-1b4c-4868-9a9a-1473c4c820de.png", 
    "Path": "/1/2", 
    "Id": 2
  }, 
  "result": 1
}
